# geek-collector

긱뉴스(GeekNews)에서 기술 글을 자동으로 수집해 옵시디언에 저장하는 도구입니다.

AI 관련 글은 자동으로 걸러내고, 추천+댓글 높은 순으로 최대 20개만 골라서 저장합니다.

---

## 이런 게 됩니다

매일 긱뉴스 들어가서 괜찮은 글 찾는 게 번거로울 때, Claude Code에 말 한마디면 됩니다.

```
"오늘 긱뉴스 수집해서 옵시디언에 정리해줘"
"어제 날짜 기준으로 긱뉴스 수집해줘"
```

반응 좋은 기술 글 20개가 옵시디언 노트로 자동 저장됩니다.

---

## 현재 버전 (v1.0.0)

| 기능 | 설명 |
|------|------|
| AI 뉴스 자동 제외 | GPT, LLM, 에이전트 등 30개 키워드 필터 |
| 추천+댓글순 정렬 | 반응 높은 글부터 저장 |
| 옵시디언 저장 | 노트별 저장 + index.md 양방향 위키링크 |
| 날짜 지정 수집 | 특정 날짜 기준으로 수집 가능 |

## 설치 방법

**Claude Code(또는 확장 프로그램)에 아래 URL을 붙여넣고 "이거 설치해줘"라고 하면 됩니다.**

```
https://github.com/airaum/geek-collector
```

---

설치 과정에서 아래 2가지는 직접 설정해야 합니다.

**1. 옵시디언 저장 경로**
- 수집한 노트를 저장할 옵시디언 폴더 경로
- 예: `D:/Obsidian/긱뉴스`

**2. 옵시디언 플러그인 설치**
- Dataview 플러그인 활성화 (대시보드 뷰용)

---

### 설치 확인

설치 후 Claude Code에서:

```
오늘 긱뉴스 수집해줘
```

옵시디언에 노트가 생기면 성공입니다.

---

## 주의사항

- 긱뉴스 사이트 구조 변경 시 수집이 안 될 수 있습니다.
- 외부 API, 유료 서비스 없이 동작합니다.
- 스크립트 실행 환경에 Python과 requests, beautifulsoup4가 필요합니다.
